HTML5 e CSS

HTML5 é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software. É a quinta versão da linguagem HTML.

CSS é um mecanismo para adicionar estilos a uma pagina web

Tipos de Seletores Uso Descrição
universal close O seletor universal * seleciona todos os elementos na página HTML. É um seletor muito genérico e deve ser usado com cuidado, pois pode afetar o desempenho do site se for usado em excesso. Ele pode ser útil para aplicar estilos básicos a todos os elementos, por exemplo, definir uma margem ou preenchimento padrão para todos os elementos na página
tipo close seleciona elementos com base no seu tipo HTML. Por exemplo, o seletor "p" seleciona todos os parágrafos no documento.
classe close seleciona elementos com base na sua classe HTML. Por exemplo, o seletor ".destaque" seleciona todos os elementos com a classe "destaque".
id close seleciona elementos com base no seu ID HTML. Por exemplo, o seletor "#cabecalho" seleciona o elemento com o ID "cabecalho".
atributo close Agrupamento por secções
pseudo-classe close Essas pseudo-classes podem ser usadas em conjunto com outros tipos de seletores CSS para criar regras de estilo mais específicas.
pseudo-elemento close Os pseudo-elementos são utilizados para adicionar e estilizar elementos fictícios em um documento HTML. Eles permitem que você selecione e estilize partes específicas de um elemento HTML que não são representadas pelo próprio elemento, como a primeira letra ou linha de um bloco de texto.
